Remindr


Description du projet

Reminder est un site web dont le but est de permettre aux utilisateurs de l'application de créer des rappels pour les différents projets, devoirs, ou attendus à rendre sur des classes, des groupes et autres, avec les détails de contexte nécessaires

Les utilisateurs peuvent créer groupes et inviter d'autres membres à le rejoindre. Une fois dans le groupe, les utilisateurs peuvent créer des reminders (rappels) qui seront visibles par tous les membres de ce groupe.


Propriété d'un groupe :
  • Nom
  • Description
  • Image

Propriété d'un reminder :
  • Titre
  • Description
  • Date de rendu
  • Couleur

Les utilisateurs peuvent se connecter en utilisant leur compte GitHub grace à Next-Auth qui permet d'utiliser différents providers tel que celui de GitHub.

Réalisé en mars 2023.
